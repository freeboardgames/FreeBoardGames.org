Checklist for publishing a new game:

1. [ ] Does not break intellectual properties of others. Check for copyright in game name, instructions, graphics, etc.
2. [ ] Playable end-to-end on desktop devices.
3. [ ] Playable end-to-end on mobile devices.
4. [ ] Has big enough touch targets (buttons, etc) on mobile.
5. [ ] Has high test coverage for the rules, and ideally some tests for the UI.
6. [ ] Shows the game over page correctly.
7. [ ] Has appropriate thumbnail, instructions, etc.
